# CSRDash

PLT CSR Dashboard with dynamic competition tabs, in-app Metrics Admin, and in-app Links Admin.

## Key updates

- Competition tabs are now driven by `Competition_Categories` and rendered dynamically.
- Competition table values/notes are stored in `Competition_Entries` and edited inline with batch saves.
- Patio Cushion Signup link is managed in-app via `App_Links` (`PATIO_CUSHION_SIGNUP`) and appears on Dashboard + Patio Signups tab.
- No redeploy needed for day-to-day metric/tab/link changes.
